Пример: кастомный заголовок, кастомные фильтры, модули модифицирующие фильтры

Source code

1)Задаётся опция "showTitleHeader: false", отключающая стандартный заголовок таблицы.
Вместо него мы подключаем свой, задав ему класс "grid-header".
2)Подключаются дополнительные модули:
new DateRangeModule(myGrid); - Инициализирует поля фильтрации для ввода даты.
new NumberRangeModule(myGrid); - Инициализирует поля фильтрации для ввода чисел.
new MultiselectModule(myGrid); - Инициализирует мультиселекты в полях фильтрации (библиотекой bootstrap-multiselect).
