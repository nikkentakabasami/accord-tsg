Пример: кастомный заголовок, кастомные фильтры, доп модули

Source code 
1)Кастомный заголовок таблицы.
Для этого задаётся опция "showTitleHeader: false", отключающая стандартный заголовок таблицы.
Вместо него мы подключаем свой, задав ему класс "grid-header".

2)Подключаем кастомные фильтры, добавив их в скрытый div filterContainer.
Для их подключения - достаточно чтобы их имена совпадали с id столбцов.

Столбец id содержит опции для быстрого выбора.
Столбцец "Выполнено" - теперь select


3)Подключаются дополнительные модули: 
new DateRangeModule(myGrid); - Инициализирует поля фильтрации для ввода даты (поля с классом "date-input").
new NumberRangeModule(myGrid); - Инициализирует поля фильтрации для ввода чисел. (поля с классом "number-input").
new MultiselectModule(myGrid); - Инициализирует мультиселекты в полях фильтрации (библиотекой bootstrap-multiselect).

Все эти модули используют bootstrap3 для работы.
