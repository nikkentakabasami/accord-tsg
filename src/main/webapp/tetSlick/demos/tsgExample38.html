Пример: Предзаданные значения фильтров

Source code 

Задать значения фильтров при создании таблицы можно несколькими способами:
1)Объявить собственный фильтр в скрытом диве filterContainer.
При этом задать на него значение.

Например:
<input name="start" class="date-input" value="19.05.2020"/> 

2)Создать hidden, с именем нужного поля, и задать значение на него.
Обычно такое используется для мультиселектов.
<input type="hidden" name="customer" value="12,76" />

3)Сгенерировать фильтр, указав при этом выбранное значение:

//Выбираем банан
fruitColumn.filterInput = accordUtils.generateSelect("fruit", dataSource.fruits, false,false,2);


4)Использовать опцию столбца initalFilterValue:

  {
  id: "durationInt",
  captionField: "duration",
  name: "Длительность",
  initalFilterValue: "24",
  width: 150
  }
