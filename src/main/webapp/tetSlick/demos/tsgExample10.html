Пример1: Простейшая таблица. Минимальная конфигурация

Source code 
let columns = [ {
	id: "title", 
	name: "Заголовок",
	width: 150 
},{ ...

let options = {
	singleRowSelectionModel: true
};				

myGrid = new TetSlickGrid("#myGrid", ds.rows, columns, options);

По умолчанию работает сортировка по столбцам. Сделайте щелчок мышью на столбцах чтобы задать столбец и направление сортировки.
Кроме того есть возможность изменять размеры столбцов, растягивая их по границам.
К самому объекту-таблице можно получить доступ через глобальный объект grid1.
Например grid1.dataLoader.data - показываемые строки; grid1.model.columns - текущие столбцы. grid1.getSortColumns() - столбцы, по которым идёт сортировка.
