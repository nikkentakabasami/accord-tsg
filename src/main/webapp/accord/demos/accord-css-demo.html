accord.css - содержит разнообразные вспомогательные стили.

accord.css содержит разнообразные вспомогательные стили. Эта панель с классом "acc-desc-panel" используется для показа текстовых пояснений. Текст в ней преформатирован.
Панель с классом "acc-code-panel" используется для показа программного кода.

acc-button-panel - Горизонтальная панель. иконки задаются через img

иконки задаются через background-image (посредством css-классов:acc-btn-add,acc-btn-cross...)
