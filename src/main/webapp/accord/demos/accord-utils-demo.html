accordUtils - содержит разнообразные вспомогательные методы

accordUtils.accordPath = "" accordUtils.random(10) = "" accordUtils.randomDate() = "" accordUtils.formatDate(today) = "" accordUtils.parseDate("05.05.2025") = ""

accordUtils.decorInput($input, options); - Добавление кнопки с иконкой программно.

все значения по умолчанию: accordUtils.decorInput($("#tf1"));
Добавление кнопки с заданной иконкой в конец input: accordUtils.decorInput($("#tf2"),{ addButton: true, buttonClasses: "acc-btn-check", placeButtonBefore: false, buttonHandler: e=>{ alert("hello"); } });

accordUtils.loadHtmlFragmentXHR

Тестируем загрузку методом loadHtmlFragmentXHR(fragmentUrl, $target, relativeToAccord = false). accordUtils.loadHtmlFragmentXHR("demos/misc/testFragment.html","#testFragment",true);

accordUtils.loadHtmlFragmentFetch

Тестируем загрузку методом loadHtmlFragmentFetch(fragmentUrl, $target, relativeToAccord = false). accordUtils.loadHtmlFragmentFetch("demos/misc/testFragment.html","#testFragment2",true) .then(result => { console.log("loaded:",result); });
