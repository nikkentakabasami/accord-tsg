Компонент waitPanel - панель ожидания

Исходники
Панель можно показывать используя методы showWaitPanel(), hideWaitPanel(). Их можно либо импортировать: import {showWaitPanel,hideWaitPanel} from 'accord/js/accord-publish.js'; Либо просто подключить к странице accord-publish.js, который делает их глобальными.
